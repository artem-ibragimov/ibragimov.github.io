CSS Grid: Гайд по решётке

CSS Grid Layout - что это такое?

Grid - изумителен, ведь он позволяет создавать сложные и гибкие макеты. Одновременно с этим Grid не прост, и мне потребовалось время, чтобы его освоить.

Поддержка браузерами

Grid - современный инструмент верски макетов, поддерживается с 2017 года почти 98 % браузеров - потрясающая поддержка, только у Flexbox немного лучше.

Модель Grid

CSS предоставляет несколько алгоритмов разметки:

  • Flow - макет по-умолчанию
  • Table - для отображения табличных данных
  • Flexbox - для отображения элементов по одной оси
  • Grid - для отображения элементов по двум осям
И первое что сбивает с толку в Grid: структура сетки - столбцы и строки - определяются в CSS! Это необычно, ведь в других макетах структура определяется дочерними DOM элементами.

Grid Макет

По-умолчанию создается один столбец и дочерние элементы располагаются на каждой строке - это считается неявной сеткой, т.к мы не задали структуру. Высота родителя определяется дочерними элементами по правилам Flow макета, и только дочерние компоненты размещаются по правилам grid сетки. Если выстоу задать явно, она будет равномерно распределена между строками

  .parent {
      display: grid;
   }
child #0

Структура сетки

Столбцы в grid задаются свойством grid-template-columns. Можно задавать в px, %, viewport units и в новых единицах fr [fraction - пропорции]. Например, grid-template-columns: 1fr 3fr; описывает два столбца:

  • первый занимает четверть [1/1+3] свободного пространства
  • второй - три четверти [3/1+3]
И если проценты и единицы длины жестко определяют размер, то fraction добавляет гибкости, позволяет растягивать и сжимать стоблцы, учитывая их содержимое.

Отличия % и fr

Столбцы, заданные fraction не сжимаются меньше минимального размера контента, т.е при малых размерах пропорции все же нарушаются, а все потому, что fraction - распределяет свободное пространство.

[Кликни на блок, чтобы сжать] 
  .parent {
      display: grid;
      grid-template-columns: 25% 75%;
   }
child 25%
child 75%

Более того fraction учитывает значение gap [расстояние между строками и столбцами в grid]. Проценты расчитываются от длины всего контейнера, 25% и 75% распределют всю длину и добавление gap переполняет контейнер. Fraction в свою очередь распределяет свободное пространство, ширину без gap

Явные и неявные строки

Grid по-умолчанию выделяет ячейку для каждого дочернего элемента, таким образом, 3 элемента в сетке с 2-мя столбцами будут размещены на 2х строках. Это неявное задание строк. Задать число строк явно можно через grid-template-rows

  .parent {
      display: grid;
      grid-template-columns: 1fr;
      grid-template-rows: 1fr;
   }
child #0

Функция repeat

Для повторяющихся значений используйте функцию repeat: grid-template-columns: repeat(5, 1fr);

Привязанность к детям

По-умолчанию алгоритм grid связывает каждый дочерний элемент с первой пустой ячейкой. Но можно привязать любые ячейки к любым дочерним элементам: свойства grid-column и grid-row определяют, какие ячейки занимает компонент. Чтобы указать диапазон, используйте номера столбцов или строк, разделенные через / [slash].

  
   .parent {
      display: grid;
      grid-template-columns: repeat(4, 1fr);
      grid-template-rows: repeat(4, 1fr);
   }

   .child {
      grid-column: 2 / 4;
      grid-row: 2 / 4;
   }
child

Отсчет ведется слева направо, сверху вниз, для обратного направления используются отрицательные значения

Grid Areas

Так, настало время поговорить о крутейшей фишке CSS GRID: разбить сетку на именнованные области и присваивать их вместо номеров столбцов и строк!

  .parent {
      display: grid;
      grid-template-columns: 2fr 5fr;
      grid-template-rows: 50px 1fr;
      grid-template-areas:
         'sidebar header'
         'sidebar main';
   }
   .sidebar {
      grid-area: sidebar;
   }
sidebar
header
main

Каждая текстовая строка ассоциируется с grid-строкой, каждое слово - со столбцом. На общем плане такой текст тоже выглядит как сетка! Дублирование sidebar на несколько строк расширяет область на эти строки.

Выравнивание

Распределение свободного пространства реализовано схожим с Flexbox алгоритмом

  • justify - выравнивание столбцов
  • align - выравнивание строк
  • content - выравнивание сетки
  • items - выравнивание DOM элементов внутри сетки 

  .parent {
      display: grid; 
   }
child #0
child #1
child #2

Есть удобное сокращение: place-content удобно указать одно значение для align-content и justify-content
